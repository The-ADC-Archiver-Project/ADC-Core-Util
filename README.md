# ADC Core

Compatible with: <img src="https://upload.wikimedia.org/wikipedia/commons/8/87/Windows_logo_-_2021.svg" alt="Windows 11" width="20"/> **&** <img src="https://upload.wikimedia.org/wikipedia/commons/3/35/Tux.svg" alt="Linux" width="20"/>

---

This repository is for OEM manufactures and people who want to open `.adc` archives without having to install the full ADC Archiver software.

After installing double click an ADC archive and it will extract in the same directory, when password protected archives are double clicked the terminal will open and ask for the corresponding password.
